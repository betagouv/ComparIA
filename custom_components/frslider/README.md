# `gradio_frslider`
<img alt="Static Badge" src="https://img.shields.io/badge/version%20-%200.0.1%20-%20orange">  

Python library for easily interacting with trained machine learning models

## Installation

```bash
pip install gradio_frslider
```

## Usage

```python

import gradio as gr
from gradio_frslider import FrSlider


example = FrSlider().example_value()

demo = gr.Interface(
    lambda x:x,
    FrSlider(),  # interactive version of your component
    FrSlider(),  # static version of your component
    # examples=[[example]],  # uncomment this line to view the "example version" of your component
)


if __name__ == "__main__":
    demo.launch()

```

### Events

| name | description |
|:-----|:------------|
| `change` | Triggered when the value of the FrSlider changes either because of user input (e.g. a user types in a textbox) OR because of a function update (e.g. an image receives a value from the output of an event trigger). See `.input()` for a listener that is only triggered by user input. |
| `input` | This listener is triggered when the user changes the value of the FrSlider. |
| `release` | This listener is triggered when the user releases the mouse on this FrSlider. |



### User function

The impact on the users predict function varies depending on whether the component is used as an input or output for an event (or both).

- When used as an Input, the component only impacts the input signature of the user function.
- When used as an output, the component only impacts the return signature of the user function.

The code snippet below is accurate in cases where the component is used as both an input and an output.

- **As output:** Is passed, passes slider value as a {float} into the function.
- **As input:** Should return, expects an {int} or {float} returned from function and sets slider value to it as long as it is within range (otherwise, sets to minimum value).

 ```python
 def predict(
     value: float
 ) -> float | None:
     return value
 ```
